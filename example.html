With Configuration

<script>
window.CookieBannerConfig = {
  message: 'We use cookies to improve your experience.',
  acceptText: 'Got it',
  rejectText: 'No thanks',
  onAccept: () => console.log('User accepted cookies'),
  onReject: () => console.log('User rejected cookies')
};
</script>
<script src="dist/cookie-banner.min.js"></script>

API

  • CookieBanner.hasConsent() - Returns true, false, or null
  • CookieBanner.accept() - Programmatically accept
  • CookieBanner.reject() - Programmatically reject
  • CookieBanner.reset() - Clear consent and show banner
  • CookieBanner.show() - Show banner
  • CookieBanner.hide() - Hide banner
